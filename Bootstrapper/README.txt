Как собрать инсталятор?
Для этого установщика нужно скачать дополнительные зависимости.
dotNetFx40_Full_x86_x64.exe - 4.0 full fraemwork
NDP40-KB2468871-v2-x86.exe - расширение для windows xp необходимое для работы с асинхронными тасками.
Эти компоненты нужно поместить в папку Redist.

При этом, если у вас собирется проект R-173, то собрав этот инсталятор у вас каскадом собирется и Installer и Bootstrapper.
Просто пересобирите проект Bootstrapper.
